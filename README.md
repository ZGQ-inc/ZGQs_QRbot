# 开始使用

打开 `QRSpeed` ，输入你的**小号**的 `账号`  `密码` ，勾选 `记住密码`  `自动登录` ，点击 `登入` 。

转到 `插件列表` ，勾选 `QRDICPro` 。

长按 `QRDICPro` 一栏，勾选 `授权skey`  `授权pskey` 。

**为防止进程被系统误杀**，点击 `后台权限` ，点击 `电池优化白名单` ，在弹出的对话框点击 `确认` 。

返回，点击 `QRDICPro` 一栏，设置 `主人` 为自己的**大号**。

接下来的步骤可供选择。

方法①：下载词库文件 `dicpro.txt` 到本地相对路径 `内部存储/QR/QRDic/` 文件夹下。

如果无法成功移动文件，可以选用方法②。

方法②：打开 `QRDICPro` ，点击右边的编辑按钮，填入


同理返回到主界面，点击左上角 `侧边栏` ，点击 `后台权限` ，点击 `电池优化白名单` ，在弹出的对话框点击 `确认` 。

返回，转到 `群列表` ，勾选需要启用的群。

# 常见问题

- **对系统误杀进程的解决办法**

由于系统软件供应商自身的问题，主进程经常被系统电池优化服务强制停止，可以在 `QRSpeed` 的 `设置` 中选择勾选 `双进程守护`  `不息屏`  `锁屏活动`  `常驻后台`  `悬浮窗保活` ，还可以参考[Don’t kill my app!](https://dontkillmyapp.com/)进行一系列设置，确保进程在后台稳定运行。

- **对腾讯风控管制的解决方法**

暂无根治方法。
